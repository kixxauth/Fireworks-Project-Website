Customized software is complicated and expensive.

Right?

But maybe it doesn't have to be.

We've found a way to make it simple and affordable by thinking a little differently.

We imagined an artisan software bakery,

where each batch of software could be crafted with personal attention.

Then we borrowed some ideas from the assembly lines used to build the first affordable automobiles.

Inspired as only geeks can be, we designed our own digital nuts and bolts for building interchangeable software components.

As it turns out, interchangeable parts have been the key to making affordable goods since the industrial revolution.

Software made from interchangeable parts will enable you to afford customized information systems that were once only available to very large companies.

You'll be able to add the software components you need, without getting stuck with the ones you don't.
