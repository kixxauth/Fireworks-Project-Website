Hackers' and Hitchhikers' Guide to The Fireworks Project
========================================================

This folder contains all HTML files for the 'Hackers' and Hitchhikers' Guide'.
The mime type for this folder is set to `text/html` in `app.yaml` and so any
files served from here will be served with the `text/html` mime type HTTP
response header. Do **not** put CSS, JS, or image files in this folder.

All CSS, JS, and other files should be served from the appropriate directory in
`google-app-engine/static/`.
