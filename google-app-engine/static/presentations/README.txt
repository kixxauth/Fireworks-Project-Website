Presentations Documents
=======================

This directory contains a random collection of content served statically as
online media for presentations and slide shows. Only the HTML portions of presentations
are permited in this directory. Other file types must be put in the appropriate `js/`,
`css/` or `img/` directories.

This folder can be referenced in HTML by an absolute URL like:

    `http://www.fireworksproject.com/presentations/`
